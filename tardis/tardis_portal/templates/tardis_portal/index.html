MyTardis Data Store (beta)

This is a system designed to store data from the IAGO Share (lsp-merc-09.its.monash.edu) into the LaRDS petabyte store. Experiments stored in MyTardis are private by default, but can be shared via the web directly with other users/groups or by making data public.


For access and instructions contact Steve Androulakis (steve.androulakis@monash.edu)
